## ðŸŽ¨ Adding New Company Logos

### To Add a New Company:

1. **Save the company logo:**
   - Place the logo image in your `Logo/` folder

## ðŸŽ›ï¸ Customization Options

### Adjust Animation Speed
In `style.css`, find `.logo-track` and change the animation duration:

```css
.logo-track {
    animation: scroll 30s linear infinite; /* Change 30s to your preferred speed */
}
```
- Faster: Use lower number (e.g., 20s)
- Slower: Use higher number (e.g., 45s)

### Change Logo Size
In `style.css`, find `.logo-item`:

```css
.logo-item {
    height: 80px; /* Adjust this value */
}

.logo-item img {
    max-width: 200px; /* Adjust this value */
}
```

### Adjust Spacing Between Logos
In `style.css`, find `.logo-track`:

```css
.logo-track {
    gap: 80px; /* Adjust spacing between logos */
}
```

### Change Background Color
In `style.css`, find `.trusted-by-section`:

```css
.trusted-by-section {
    background: linear-gradient(135deg, #f8f9fa 0%, #e9ecef 100%);
    /* Change to your preferred gradient or solid color */
}
```

## ðŸ› Troubleshooting

### Issue: Logos Not Appearing
**Possible Causes:**
1. File paths are incorrect (check case-sensitivity)
2. Logo files don't exist in the `Logo/` folder
3. JavaScript file not loaded properly

**Solution:**
- Open browser console (F12) and check for errors
- Verify all file paths in the `trustedCompanies` array
- Ensure `trusted-by-carousel.js` is in the correct location

### Issue: Animation Not Smooth
**Possible Causes:**
1. Not enough logo duplicates
2. Animation speed too fast

**Solution:**
- The script automatically creates duplicates
- Try increasing animation duration in CSS
- Add more companies to fill the screen width

### Issue: Logos Too Small/Large
**Solution:**
- Adjust `.logo-item` height in CSS
- Adjust `.logo-item img` max-width in CSS
- Optimize your logo file dimensions before uploading

### Issue: Section Not Visible
**Solution:**
- Check if the section is placed correctly in HTML
- Verify CSS file is loading (check Network tab in browser)
- Clear browser cache and reload
